    Remarks

    TODO: Возможно будет лучше если алгоритм будет выполняться полностью изолированно от Links на этапе сжатия. А именно будет создаваться временный список пар необходимых для выполнения сжатия, в таком случае тип значения элемента массива может быть любым, как char так и ulong. Как только список/словарь пар был выявлен можно разом выполнить создание всех этих пар, а так же разом выполнить замену.
